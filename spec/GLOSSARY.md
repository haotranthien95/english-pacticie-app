# Glossary

## Technical Terms

**API (Application Programming Interface)**
A set of rules and protocols that allows different software applications to communicate with each other.

**Authentication**
The process of verifying the identity of a user or system.

**Authorization**
The process of determining what permissions a verified user has.

**Backend**
The server-side of the application that handles data processing, business logic, and database operations.

**CI/CD (Continuous Integration/Continuous Deployment)**
Automated processes for testing and deploying code changes.

**Endpoint**
A specific URL in an API where a client can access a particular resource or functionality.

**JWT (JSON Web Token)**
A compact token format for securely transmitting information between parties.

**Mobile App**
The client-side application running on iOS or Android devices.

**RESTful API**
An API architectural style that uses HTTP requests to access and manipulate data.

**Schema**
The structure and organization of a database, including tables, fields, and relationships.

## Product Terms

**Achievement**
A milestone or accomplishment that users can unlock by completing specific tasks or reaching goals.

**Daily Challenge**
A single exercise or set of exercises available each day to encourage consistent practice.

**Exercise**
An individual practice activity (question, drill, or task) within a lesson.

**Gamification**
The use of game-like elements (points, badges, levels) to increase user engagement and motivation.

**Learning Path**
A personalized sequence of lessons and exercises tailored to a user's level and goals.

**Lesson**
A structured learning unit that covers a specific topic or skill.

**Proficiency Level**
A categorization of language ability: Beginner, Intermediate, or Advanced.

**Progress Tracking**
The system for monitoring and displaying a user's learning activities and improvements.

**Spaced Repetition**
A learning technique that presents content at increasing intervals to optimize retention.

**Streak**
The number of consecutive days a user has engaged with the app.

## Learning Terms

**Comprehension**
The ability to understand written or spoken language.

**Grammar**
The rules that govern the structure of sentences and language.

**Listening**
The skill of understanding spoken language.

**Pronunciation**
The way words are spoken, including sounds and stress patterns.

**Reading**
The skill of understanding written language.

**Speaking**
The skill of producing spoken language.

**Vocabulary**
The set of words known and used by a person.

**Writing**
The skill of producing written language.
